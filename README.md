🚀 Excited to share my latest project: Text Notes Keeper!

I developed this modern Android application to provide a seamless and efficient experience for managing daily thoughts, study notes, and reminders. The goal was to build a tool that balances high performance with a clean, intuitive UI.

Key Technical Highlights:

Modern UI: Built entirely with Jetpack Compose for a reactive and smooth user interface.

Local Persistence: Implemented Room Database for fast and reliable data storage.

Architecture: Followed MVVM pattern and Clean Architecture principles to ensure code scalability and maintainability.

Smart Features: Integrated AI-powered text rephrasing and diacritization (تشكيل) using Groq API.
